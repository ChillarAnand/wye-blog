## Python Express Blog

## Setup

Clone repo to your local system.

```sh
git clone https://github.com/pythonindia/wye-blog.git
```

Install [lektor]() in your system.

```sh
pip install lektor
```

Start server using

```sh
lektor server
````

Go to browser and open http://127.0.0.1:5000/
